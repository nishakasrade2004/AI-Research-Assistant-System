AI Research Assistant System

Description
AI Research Assistant System is a web-based application that uses Artificial Intelligence to generate structured and meaningful content based on user input topics. It helps users quickly understand concepts with organized information, detailed explanations, and features like Read More and PDF download.

Features
-  AI-based content generation  
-  Structured topic explanation  
-  Read More (detailed subtopics)  
-  PDF download option  
-  User-friendly interface  

Technologies Used
- Python  
- Flask / Streamlit  
- MySQL / PostgreSQL  
- OpenAI API  
- HTML, CSS  

How to Run
1. Install dependencies  
2. Run the main file (app.py)  
3. Open in browser
